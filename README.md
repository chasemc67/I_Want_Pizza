### Pizza Ordering Voice Bot

A voice bot built using API.ai to order pizza.  
This was built at Hack the North in 2016.

<a href="http://www.youtube.com/watch?feature=player_embedded&v=nNC94nuwxQw
" target="_blank"><img src="http://img.youtube.com/vi/nNC94nuwxQw/0.jpg" 
alt="IMAGE ALT TEXT HERE" width="240" height="180" border="10" /></a>


Features include:
* Remembering your last order
* Customization of orders
* Ordering multiple pizzas


#### Usage:
```
npm install  
npm start  
```
